## 🚀 FEATURED DEVOPS PROJECTS

### ✅ Job Queue Worker System (Redis + FastAPI + Kubernetes + HPA)
A Redis-backed job processing system built to showcase Kubernetes scaling + failure recovery.

**What it proves**
- Durable queue processing (no silent drops)
- Stateless API + autoscaled worker deployments
- Redis StatefulSet + PVC (stateful workload)
- HPA scaling under real CPU load
- Requeue + stuck job recovery patterns

**Kubernetes demo commands**
```bash
kubectl -n job-system get pods
kubectl -n job-system get hpa
kubectl top pods -n job-system
watch -n 2 "kubectl -n job-system get hpa; echo '---'; kubectl -n job-system get pods -l app=worker"
````
